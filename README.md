# SISTEMA DE GESTION DE CALIDAD
Requerimientos de instalacion 

```
1. Instalacion de [PYTHON](https://www.python.org/community-landing/) version 3.8.8 recomendada. 
2. Instalacion de [MYSQL](https://www.digitalocean.com/community/tutorials/how-to-install-mysql-on-ubuntu-18-04) guia de instalacion para ubuntu. 
3. Instalacion de [PIP](https://pip.pypa.io/en/stable/) en la version mas estable.
4. Instalacion de [VIRTUALENVWRAPPER](https://pypi.org/project/virtualenvwrapper/), disponible para linux, windows o Mac Os.
5. Instalacion de [GIT](https://git-scm.com/downloads) version estable.
```
